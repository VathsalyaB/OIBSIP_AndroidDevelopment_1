# Unit Converter App

##  Overview

The Unit Converter App is an Android application developed using Java and XML that converts values across multiple measurement categories through a simple and user-friendly interface.

##  Objective

To develop a unit conversion application capable of converting values between different units while providing an intuitive and efficient user experience.

##  Features

* Multiple conversion categories
* Real-time unit conversion
* Easy-to-use interface
* Input validation
* Accurate conversion calculations
* Clean and responsive UI

##  Tools & Technologies

* Android Studio
* Java
* XML
* Android SDK

##  Steps Performed

1. Designed the application interface using XML layouts.
2. Implemented category selection.
3. Developed conversion logic for different measurement units.
4. Added validation for user inputs.
5. Tested conversion accuracy and application performance.

##  Outcome

Successfully developed a unit converter application capable of performing accurate conversions across multiple unit categories with a simple and responsive interface.
